When used , please watch README.md first !

We suggest you should use ES5 grammer .

Even though , We had used babel module to translate ES6 grammer . But in order to reduce output size , some ES6 feather's realization code , like Promise 、Map 、Set , would not be contained in finnal code .

So , you should note these pointers :

1、 Try to use ES5 grammer as much as you can ;

2、 Do some ES6 compatible ways in your personal program , like babel-polyfill 、es6-shim and so on .
